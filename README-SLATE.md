# Slate Editor

**Write fast. Read beautifully.**

Slate Editor is a fast, focused writing environment for Markdown, RTF, and plain text on Windows.  
It’s designed to feel calm, responsive, and precise—so you can stay in flow and trust what you see.

---

## ✨ Overview

Slate is built around a simple idea:

> Writing tools should get out of the way.

Whether you're working in Markdown, rich text, or plain text, Slate gives you a clean surface with high-quality rendering and minimal friction.

---

## ⚡ Key Features

- **Markdown-first experience**  
  Clean syntax highlighting, predictable rendering, and a seamless editing flow.

- **Multi-format support**  
  Work with:
  - `.md` (Markdown)
  - `.rtf` (Rich Text Format)
  - `.txt` (Plain text)

- **Fast and responsive**  
  Designed for low latency and smooth interaction, even with large files.

- **Beautiful reading mode**  
  Carefully tuned typography and spacing for long-form readability.

- **Cappuccino themes**  
  Warm, low-contrast **dark and light themes** designed for extended use.

- **Distraction-free interface**  
  Minimal UI, no clutter, no unnecessary panels.

---

## 🎯 Design Principles

Slate is guided by a few core principles:

- **Clarity over features**  
  Every element should justify its presence.

- **Speed as a feature**  
  Performance is not optional—it’s foundational.

- **Consistency across formats**  
  Markdown, RTF, and plain text should feel like one coherent system.

- **Calm visual design**  
  The interface should reduce cognitive load, not add to it.
